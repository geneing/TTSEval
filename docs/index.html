TTS Comparison

This is a small test of a few popular recent TTS models. I wanted to see how good these systems are for generating audiobooks.

I used default weights and parameters. Code is here

I provided kokoro sample to the other three models as input for zero-shot cloning. I'm not testing voice cloning, but it's easier to compare output when voice is similar for the samples.

Test 1. Alignment test.

Testing for alignment issues on a text with repeated word patterns.

Notice that fish-speech skips some words in the middle, which is a common problem when using LLM based TTS. Chatterbox has a strange noise at the end of the audio, which is also a common problem with LLM based TTS. Mega_tts3 makes an unnatural pause before "then".

Text

She sells seashells on the seashore.
The shells she sells are seashells, I’m sure, so if she sells seashells on the seashore, then I’m sure she sells seashore shells.

Test 3. Emotive reading.

Testing on an emotive text. Does the TTS render emotions well?

In principle, all four TTS systems should be able to read this text with the right intonation and emotion. Does it have the right balance of emotion, clarity, and expressiveness for reading audiobooks? Too much emotion can make it sound unnatural, while too little can make it sound flat.
